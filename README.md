Installation
--------------
* Clone this project
```sh
git clone https://github.com/miaoyinjun/spring_boot_test.git
```

* Build Application
```sh
cd spring_boot_test
mvn clean package
```
* Run Application
```sh
cd target
java -jar spring_boot_test.jar
```
Try it
--------------
- Now open your browser: http://localhost:8080/
